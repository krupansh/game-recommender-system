# Game Recommendation System

A collaborative filtering-based recommendation system for suggesting games based on user preferences.

## Overview

This project implements a recommendation engine using collaborative filtering techniques. It analyzes user-game interactions and generates personalized recommendations.

The system also integrates the RAWG API to fetch game metadata.

## Tech Stack

- Python
- FastAPI
- Streamlit
- Pandas
- Scikit-learn

## Features

- Collaborative filtering recommendation engine
- CRUD API for managing game data
- Interactive dashboard
- External game metadata integration (RAWG API)
